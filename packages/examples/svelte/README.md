# Simple Table — Svelte examples

Runnable Svelte 4 / Svelte 5 (Vite) demos for the [`@simple-table/svelte`](https://www.npmjs.com/package/@simple-table/svelte) data grid. Compatible with SvelteKit and any Vite-powered Svelte app. These demos power the live sandboxes on [simple-table.com](https://www.simple-table.com) and the one-click StackBlitz links from the docs.

## Run locally

```bash
pnpm install
pnpm --filter examples-svelte dev
```

## What's inside

Real-world demos that showcase Simple Table on Svelte 5 (with runes) and Svelte 4:

- **Quick start**, **Sales**, **CRM**, **HR**, **Music**, **Manufacturing**, **Infrastructure**, **Billing**
- Feature focuses: virtualization, column pinning, resizing, reordering, visibility, nested headers, row grouping, inline cell editing, custom renderers via Svelte components, themes, pagination, infinite scroll

## Use this in your project

```bash
npm install @simple-table/svelte
```

See the [Quick Start guide](https://www.simple-table.com/docs/quick-start) and the [Svelte framework hub](https://www.simple-table.com/frameworks/svelte) for up-to-date usage instructions.
