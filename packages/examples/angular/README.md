# Simple Table — Angular examples

Runnable Angular 17+ (standalone components) demos for the [`@simple-table/angular`](https://www.npmjs.com/package/@simple-table/angular) data grid. These demos power the live sandboxes on [simple-table.com](https://www.simple-table.com) and the one-click StackBlitz links from the docs.

## Run locally

```bash
pnpm install
pnpm --filter examples-angular dev
```

## What's inside

Real-world demos that showcase Simple Table on Angular standalone components with signals:

- **Quick start**, **Sales**, **CRM**, **HR**, **Music**, **Manufacturing**, **Infrastructure**, **Billing**
- Feature focuses: virtualization for 1M+ rows, column pinning, resizing, reordering, visibility, nested headers, row grouping, inline cell editing, custom renderers via Angular components, themes, pagination, infinite scroll, programmatic control. AOT-friendly, Ivy-compatible, TypeScript strict mode

## Use this in your project

```bash
npm install @simple-table/angular
```

See the [Quick Start guide](https://www.simple-table.com/docs/quick-start) and the [Angular framework hub](https://www.simple-table.com/frameworks/angular) for up-to-date usage instructions.
