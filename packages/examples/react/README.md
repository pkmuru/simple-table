# Simple Table — React examples

Runnable React (Vite) demos for the [`@simple-table/react`](https://www.npmjs.com/package/@simple-table/react) data grid. Each demo also powers an interactive sandbox on [simple-table.com](https://www.simple-table.com) and a one-click StackBlitz link from the docs.

## Run locally

```bash
pnpm install
pnpm --filter examples-react dev
```

The dev server runs on http://localhost:5200.

## What's inside

Real-world demos that showcase Simple Table on React 18+:

- **Quick start**, **Sales**, **CRM**, **HR**, **Music**, **Manufacturing**, **Infrastructure**, **Billing**
- Feature focuses: virtualization, column pinning, column resizing, column reordering, column visibility, nested headers, row grouping, row selection, cell editing, custom cell/header/footer renderers, themes, pagination, infinite scroll, programmatic control, external sort/filter, and more

## Use this in your project

```bash
npm install @simple-table/react
```

See the [Quick Start guide](https://www.simple-table.com/docs/quick-start) and the [React framework hub](https://www.simple-table.com/frameworks/react) for up-to-date usage instructions.
