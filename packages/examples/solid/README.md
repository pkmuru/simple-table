# Simple Table — Solid examples

Runnable Solid.js (Vite) demos for the [`@simple-table/solid`](https://www.npmjs.com/package/@simple-table/solid) data grid. Compatible with Solid Start and any Vite-powered Solid app. These demos power the live sandboxes on [simple-table.com](https://www.simple-table.com) and the one-click StackBlitz links from the docs.

## Run locally

```bash
pnpm install
pnpm --filter examples-solid dev
```

## What's inside

Real-world demos that showcase Simple Table with Solid signals (fine-grained reactivity, no virtual DOM diff):

- **Quick start**, **Sales**, **CRM**, **HR**, **Music**, **Manufacturing**, **Infrastructure**, **Billing**
- Feature focuses: virtualization, column pinning, resizing, reordering, visibility, nested headers, row grouping, inline cell editing, custom renderers via Solid components, themes, pagination, infinite scroll

## Use this in your project

```bash
npm install @simple-table/solid
```

See the [Quick Start guide](https://www.simple-table.com/docs/quick-start) and the [Solid framework hub](https://www.simple-table.com/frameworks/solid) for up-to-date usage instructions.
